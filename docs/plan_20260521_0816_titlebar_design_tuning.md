## 1. Problem Summary (문제 정의)
- 창 상단 타이틀바에서 브랜드 이름인 "CostBIM" 텍스트와 다이아몬드 로고 아이콘의 수직 위치가 불균형하여 프리미엄 감성이 떨어짐.
- 최대화 버튼(`⬜`)이 축소(`─`)나 닫기(`✕`) 버튼에 비해 폰트 크기가 `9`로 너무 작게 매핑되어 시각적 비대칭 유발 및 조작 피로 증가.
- 기존의 평면 다이아몬드 보석 모양 아이콘 대신, CostBIM 솔루션의 3D 공간 기하 연산 및 적산(Cost) 성격을 웅변할 수 있는 고유하고 심플한 3D 입체 큐브형 로고가 필요함.

## 2. Design Summary (설계 요약)
- **목적**: 상단 타이틀 영역의 완벽한 픽셀 매칭 수직 정렬과 3D 큐브 로고화 및 제어 버튼 스케일 대칭 조율.
- **입출력 및 예외 처리**:
  - **입력**: WPF 레이아웃 로딩 및 렌더링.
  - **출력**: 완벽히 밸런스가 잡힌 상단 타이틀바 UI 렌더링.
- **주요 모듈**:
  - `MainWindow.xaml`의 TitleBar Border 영역 (`Grid.Row="0"`):
    - `Path` (Isometric 3D Cube Path 정의).
    - `TextBlock` ("CostBIM" 텍스트 픽셀 오프셋 마진 및 정가운데 정렬).
    - `Button` (Maximize 버튼 `FontSize`를 11.5로 정렬).

## 3. Implementation Plan (구현 세부 계획)
1. **[3D Isometric 로고 아이콘 디자인]**
   - 윗면, 좌측면, 우측면이 세밀히 쪼개져 3차원 입체감을 자아내는 정밀 벡터 Path 주입:
     `Data="M6,1.5 L12,4.5 L6,7.5 L0,4.5 Z M0,5.5 L6,8.5 L6,13.5 L0,10.5 Z M6,8.5 L12,5.5 L12,10.5 L6,13.5 Z"`
     크기는 콤팩트한 `Width="12" Height="12"`, 색상은 세련된 인디고 `#6366F1` 유지.
2. **[수직 정가운데 오프셋 정렬]**
   - 로고 `Path`와 `TextBlock`에 `VerticalAlignment="Center"`를 명시하고, 베이스라인 편차를 잡기 위해 `Margin` 세부 조율 (텍스트 마진 `Margin="0,0,0,0.5"`, 로고 마진 `Margin="0,0,8,0"`).
3. **[제어 버튼 밸런싱]**
   - 최대화 버튼의 `FontSize="9"` 속성을 `FontSize="11.5"`로 수정하여 폰트 렌더링 스케일 복원.

## 4. Expected Behavior (동작 시나리오)
- **로고**: 창 상단 좌측에 정밀하게 각인된 심플한 보라색 3D 큐브 로고가 CostBIM 텍스트와 완벽한 상하 대칭 수평선상에 놓임.
- **버튼**: ─, ⬜, ✕ 세 버튼이 동일한 볼륨감과 선 굵기를 지니며, 마우스 호버 시 동일한 영역 반응성을 유지함.
